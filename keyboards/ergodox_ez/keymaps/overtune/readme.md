# The Default Ergodox Layout

Compile

```bash
qmk compile -kb ergodox_ez -km overtune
```

```bash
qmk compile -kb ergodox_ez -km overtune
```

Flash

```bash
qmk flash -kb ergodox_ez -km overtune
```

Set default keymap

```bash
qmk config user.keymap=overtune
```

Set default keyboard

```bash
qmk config user.keyboard=planck/rev6
```
